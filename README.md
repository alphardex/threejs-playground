# h5-boilerplate-vue-ts

目录结构：

- apis: 各种接口请求函数
- assets: 图片素材
- components: 组件
- consts: 常量，如配置
- hooks: 钩子，即可复用的函数
- mock: 模拟请求数据
- router: 路由
- types: 类型
- utils: 工具，如提示、请求等
- views: 视图

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

